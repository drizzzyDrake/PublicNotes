L'**opcode** è la parte dell'istruzione che specifica quale tipo di operazione deve essere eseguita dal processore. L'**opcode** è codificato in una sezione specifica dell'istruzione e può rappresentare operazioni come addizione (`add`), sottrazione (`sub`), o istruzioni di salto (`jal`), tra le altre.

---

**Ad esempio:**

```csharp
add x3, x1, x2 // "add" è l'opcode e indica l'addizione
```

In alcune istruzioni più complesse, l'**opcode** può anche specificare una **sotto-operazione** (ad esempio, una variante dell'operazione principale), che viene selezionata in base ai bit aggiuntivi nell'istruzione.
