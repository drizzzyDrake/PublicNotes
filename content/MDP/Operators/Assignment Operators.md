Usati per assegnare valori alle variabili.

|Operatore|Equivalente a|Esempio|
|---|---|---|
|`=`|Assegnazione|`a = 10`|
|`+=`|`a = a + b`|`a += 5 // a = 15`|
|`-=`|`a = a - b`|`a -= 5 // a = 5`|
|`*=`|`a = a * b`|`a *= 2 // a = 20`|
|`/=`|`a = a / b`|`a /= 2 // a = 5`|
|`%=`|`a = a % b`|`a %= 3 // a = 1`|
