Operano a livello di bit.

| Operatore | Descrizione      | Esempio (`a = 5 (0101)`, `b = 3 (0011)`) |
| --------- | ---------------- | ---------------------------------------- |
| `&`       | AND bitwise      | `a & b // 1 (0001)`                      |
| \|        | OR bitwise       | `a \| b // 7 (0111)`                     |
| `^`       | XOR bitwise      | `a ^ b // 6 (0110)`                      |
| `~`       | NOT bitwise      | `~a // -6`                               |
| `<<`      | Shift a sinistra | `a << 1 // 10 (1010)`                    |
| `>>`      | Shift a destra   | `a >> 1 // 2 (0010)`                     |
