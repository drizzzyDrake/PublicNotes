Usati per aumentare o diminuire un valore.

| Operatore | Descrizione     | Esempio (`a = 10`)           |
| --------- | --------------- | ---------------------------- |
| `++a`     | Pre-incremento  | `++a // 11`                  |
| `a++`     | Post-incremento | `a++ // 10 (poi diventa 11)` |
| `--a`     | Pre-decremento  | `--a // 9`                   |
| `a--`     | Post-decremento | `a-- // 10 (poi diventa 9)`  |
