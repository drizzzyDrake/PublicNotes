Usati per combinare condizioni booleane.

| Operatore | Descrizione                    | Esempio (`x = true`, `y = false`) |
| --------- | ------------------------------ | --------------------------------- |
| `&&`      | AND logico (Entrambi veri)     | `x && y // false`                 |
| \| \|     | OR logico (Almeno uno vero)    | `x \| \| y // true                |
| `!`       | NOT logico (Inverte il valore) | `!x // false`                     |
