Restituiscono `true` o `false` a seconda della condizione.

| Operatore | Descrizione       | Esempio (`a = 10`, `b = 5`) |
| --------- | ----------------- | --------------------------- |
| `==`      | Uguale            | `a == b // false`           |
| `!=`      | Diverso           | `a != b // true`            |
| `>`       | Maggiore          | `a > b // true`             |
| `<`       | Minore            | `a < b // false`            |
| `>=`      | Maggiore o uguale | `a >= b // true`            |
| `<=`      | Minore o uguale   | `a <= b // false`           |
