Il **core del linguaggio** (o **core di un linguaggio di programmazione**) è l'insieme di **caratteristiche fondamentali** che definiscono il linguaggio stesso e che sono **indipendenti da librerie esterne o API aggiuntive**. In altre parole, il core rappresenta la base su cui si costruisce tutto il resto. Per Java, come per altri linguaggi, il core include i concetti e le funzionalità essenziali che ti permettono di scrivere e eseguire un programma.

**Contiene:**

| **Elemento**                  | **Descrizione**                                                |
| ----------------------------- | -------------------------------------------------------------- |
| **Sintassi di base**          | Regole per scrivere codice (dichiarazioni, metodi, classi)     |
| **Tipi di dati primitivi**    | `int`, `double`, `boolean`, `char`, etc.                       |
| **Controllo del flusso**      | `if`, `else`, `for`, `while`, `switch-case`                    |
| **Orientamento agli oggetti** | Classi, oggetti, incapsulamento, ereditarietà, polimorfismo... |
| **Gestione delle eccezioni**  | `try`, `catch`, `throw`, `throws`                              |
| **Metodi e funzioni**         | Scrittura e utilizzo di metodi                                 |
| **Librerie standard**         | Pacchetti come `java.lang`, `java.util`, `java.io`             |

Non contiene ad esempio: frameworks, librerie esterne, API Web.
