Un **programma** è un file passivo memorizzato in **memoria secondaria** (disco). Contiene solo il **codice** e i **dati** necessari a svolgere un certo compito, ma **non è in esecuzione**. Esempi concreti possono essere: un file `.exe`, un binario Linux, un’applicazione. (Leggi anche la definizione di [[Process|processo]])

**Caratteristiche principali:**

- È statico
- Non ha uno stato di esecuzione
- Non usa CPU o RAM finché non viene avviato
